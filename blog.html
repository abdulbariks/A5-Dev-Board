Question-1: What are the different ways to select an element in the DOM?

Different ways to select an element in the DOM are: getElementById, getElementsByClassName, getElementsByTagName, getElementsByName, querySelector, querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the HTML content inside an element. innerText: Gets or sets the visible text of an element. textContent: Gets or sets the full text content, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events for multiple child elements, improving performance and simplifying code management.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event triggered on a child element propagates upward through its parent elements until it reaches the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add, and remove elements using JavaScript, you can use document.createElement() to create an element, parentElement.appendChild() to add it, and parentElement.removeChild() to remove it.
